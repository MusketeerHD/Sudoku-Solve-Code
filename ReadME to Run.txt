Cara menggunakan source code

1. Source Code Berupa file dengan extension, .ipynb 
   yang umumnya bisa dibuka dengan google collabs dan web interaktif sejenisnya.

2. File perlu diupload ke google collabs sebagai notebook baru(Upload notebook), setelah itu perlu
   mengupload file dataset ke folder (symbol folder yang biasa ada di tab kiri).

3. Dataset bisa didapatkan di tautan berikut, "https://www.kaggle.com/datasets/bryanpark/sudoku/code"
   ,karena ukuran nya melebihi kapasitas dari BIG (50Mb), maka kami tidak bisa mencantumkannya disini.

4. Setelah selesai mengupload Dataset, Maka tahap selanjutnya adalah memasukan puzzle di line terakhir
   formatnya adalah 9 x 9, Cara memasukannya adalah cukup mengganti angka 0 sesuai dengan angka yang ada di puzzle yang ingin dipecahkan.

5. Sudoku puzzle bisa didapatkan di berbagai web, contohnya di tautan "https://sudoku.com/" , tidak disarankan menaruh angka acak dalam template puzzle
   dikarenakan sebuah sudoku puzzle standar perlu dikaji dan tidak bisa sembarang dibuat.

6. Setelah memasukan puzzle, maka anda bisa jalankan program tersebut, 
   untuk pertama run, program bisa memakan waktu yang lama karena perlu dilakukan  dataset training.

7. Jawaban dari puzzle akan keluar sebagai output di akhir program.

8. Jika Anda ingin memecahkan puzzle lainnya, anda cukup melakukan run di line 12 jika sudah mentrain dataset yang ada. 

9. Sekian adalah cara menjalankan Source code Sudoku puzzle solver.
